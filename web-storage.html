Web Storage Demo

The Web Storage API is broken up into two types of storage, localStorage and sessionStorage.

To add an item to either type of storage, do the following in your Web Browsers developer tools console:


localStorage.name = 'John';
sessionStorage.name = 'Mary';

Now if you look under the 'Applications' tab in the Chrome developer tools or the 'Storage' tab in the Firefox developer tools, you should see options for localStorage and sessionStorage. From here you can see the key/value pairs for each domain.

Try retrieving these values by typing the following in the console:


localStorage.name;
sessionStorage.name;

You may also add items using the following syntax:


localStorage.setItem('name', 'John');
sessionStorage.setItem('name', 'Mary');

You can Delete items by using:


localStorage.removeItem('name');
sessionStorage.removeItem('name');
